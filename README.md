## Setup

    $ git clone git@github.com:schurock/coffin_poll.git

## Deployment

Commit local changes, pull changes from GitHub and push:

    $ git add .
    $ git commit -m 'YOUR MESSAGE'
    $ git pull --rebase
    $ git push origin master
